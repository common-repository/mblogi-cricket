== Description ==

= How does MBlogi Cricket work? =

<a href="http://cric.mblogi.com" target="_blank">MBlogi Cricket</a> allows you easily to display Live Scores and statistics of Cricket on your site. The plugin can be installed and configured very easily.

Please visit <a href="http://cric.mblogi.com" target="_blank">MBlogi Cricket</a> for demo

Features include:

MBlogi Cricket keeps users with update cricket scores (ball by ball cricket score). It stems live data from cric.mblogi.com for the live, recent and fixture on the International and Non-international cricket,  and displays it on your blog!

1. Full Score Board: View Match full score board of every match. If a match is live/ongoing then scoreboard will be refresh every 20secs providing you an up to date info.

2. Ongoing Series: It displays ongoing cricket series matches in detail.

3. Archive: This option provides you all the cricket records and the scoreboards from 1877 (first international match) till to date. 

4. Widget: 
   I. Current: It  displays short details of ongoing matches with auto refresh for every 20 seconds.
   II. Recent: This displays 4 recent matches 
   III: Fixtures : It will display 4 upcoming matches
   
Upcoming Features:
1. Player profiles with full statistics
2. Team Ranking and details
And many more..,


Note: You'll need an MBlogi Cricket API key to use it. Get a free API KEY from <a href="http://cric.mblogi.com/mblogi-cricket-api/" target="_blank">MBlogi Cricket API</a>. In free API account you'll get 5000 API calls per month. 


== Installation ==

1. Upload the 'MBlogi Cricket' directory to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Get an free api key from <a href="http://cric.mblogi.com" target="_blank">MBlogi Cricket</a>
4. Enter the API KEY in your MBlogi Cricket setting page.
5. Place the [mblogi_cricket_archive] shortcode in a page for Cricket Archives.
6. Place the [mblogi_cricket_current_series] shortcode in a page for Cricket Ongoing Series.
7. Place the [mblogi_cricket_fullscore] shortcode in a page for Cricket match full score board.
8. Place Live Crikcet widget in widget area
